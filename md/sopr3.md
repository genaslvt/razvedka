# 3. Мониторинг и алерты

**Стек:** Prometheus + Grafana

**Проверки:**

- HTTP 200 (каждые 30 сек)
- Время ответа API (≥500 мс → warning)
- Свободное место на диске (<10% → critical)

**Алерты в Telegram:**

- Падение системы (P1)
- 5xx ошибок >10 за 5 мин
- Redis недоступен

**Дашборды:**

- Общая нагрузка
- Ошибки по эндпоинтам
- Активные сессии
