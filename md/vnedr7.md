# 7. Модуль оплаты и документов

**Счета:**

- Генерация PDF + XLS
- Статусы: `ожидает` → `оплачен` → `просрочен`

**УПД:**

- Формируется после оплаты и отгрузки
- Отправка через ЭДО (Диадок / СБИС / Астрал)

**Статусы оплаты:**

- [ ] Не оплачен
- [ ] Частично
- [x] Оплачен
- [ ] Отменен
