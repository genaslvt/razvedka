# 4. Модуль лотов

**Поля карточки:**

- `Название` (string, ≤255)
- `Артикул` (string, ≤50)
- `Цена для участников` (float, 2 знака)
- `MOQ` (integer)
- `Дедлайн` (datetime)
- `Статус` (active | closed | cancelled)

**Действия организатора:**

- Создание / копирование / архив
- Массовый импорт XLSX
- Принудительная отмена
