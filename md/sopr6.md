# 6. Работа с логами

**Срок хранения:** 90 дней

**Формат:** JSON (структурированный)

**Фильтрация:**

- По `user_id`
- По `action` (create_lot, close_purchase)
- По `timestamp`

**Обязательный аудит:**

| Действие | Кто логирует |
|----------|--------------|
| Изменение цены лота | Система |
| Ручное закрытие закупки | Система |
| Выгрузка счетов | Система |
| Вход админа | Система |

**Доступ к логам:** только суперадмин и техподдержка
