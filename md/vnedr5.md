# 5. Модуль заявок

## Параметры лота (отображение для покупателя)

| Параметр | Значение |
|----------|----------|
| Товар | Бумага Снегурочка А4, 500 листов |
| Цена (с НДС 20%) | 320 ₽ |
| Минимальный заказ (MOQ) | 10 пачек |
| Шаг заказа | 10 пачек |
| Текущий сбор | 87 из 500 пачек (17%) |
| Дедлайн | 25.12.2025 18:00 МСК |

## Элементы управления

| Элемент | Действие |
|---------|----------|
| `+` | Увеличить количество на шаг заказа |
| `-` | Уменьшить количество (не ниже MOQ) |
| Поле ввода | Ручной ввод числа |
| `[ Подать заявку ]` | Подтверждение и отправка |
| `[ Сохранить черновик ]` | Временное сохранение |

## Правила валидации

| Условие | Ошибка |
|---------|--------|
| Количество не кратно шагу | «Значение должно быть кратно X» |
| Количество меньше MOQ | «Минимальный заказ — X единиц» |
| Количество превышает остатки | «Доступно не более X единиц» |

## Статусы заявки

| Статус | Описание | Доступные действия |
|--------|----------|---------------------|
| Черновик | Не подтверждена | Редактирование, удаление, подача |
| Активна | Подтверждена | Редактирование (до стопа), отмена |
| Зафиксирована | Сбор закрыт | Только просмотр |
| Отменена | Отозвана покупателем | Копирование, просмотр |

## Функции
* Повтор заявки из истории
* Сравнение цен с предыдущей закупкой
* Сохранение шаблонов (для частых позиций)
