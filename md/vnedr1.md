# 1. Термины и аббревиатуры

| Термин | Расшифровка |
|--------|--------------|
| **ИС** | Информационная система |
| **Компания (Организатор)** | Юридическое лицо — владелец платформы |
| **Покупатель (Участник)** | Юридическое лицо (ООО, АО, ИП, ПАО) |
| **Лот** | Товарная позиция с условиями |
| **MOQ** | Minimum Order Quantity — минимальный объем |
| **Консолидация** | Процесс сбора заявок |
| **УПД** | Универсагальный передаточный документ |
| **ЭДО** | Электронный документооборот |
