# 7. Интеграционная поддержка

**Что сопровождаем:**

- API для 1С (выгрузка/синхронизация)
- Интеграция с ЭДО (Диадок)
- Интеграция с банк-клиентом (выписки)

**Автоматика:**

- Мониторинг очередей обмена (Redis)
- Автоматический перезапуск упавших воркеров
- Уведомление об ошибке интеграции в Telegram

**Ручная синхронизация:**

- Кнопка «Запустить синхронизацию» в админке
- Почасовая retry-политика для упавших УПД
