# 8. Требования к интеграции

| Система | Протокол | Данные |
|---------|----------|--------|
| 1С | REST / file | Заказы, остатки |
| Банк-клиент | API | Статусы оплат |
| ЭДО | REST (OAuth) | УПД |
| ФНС (Dadata) | API | Проверка ИНН |

**API требования:**

- REST, JSON
- JWT-токен
- Таймаут ≤ 3 сек
- Ретри: 3 попытки
