# 10. Архитектурные требования

**Стек:**

- Фронт: React 18 + TS
- Бэк: Node.js (NestJS) / FastAPI
- БД: PostgreSQL 15
- Кеш: Redis 7
- Файлы: S3 (MinIO)
- Очереди: Bull / Celery
- Логи: ELK
- Мониторинг: Prometheus + Grafana
