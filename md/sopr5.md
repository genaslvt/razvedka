# 5. Администрирование пользователей

**Заведение юр. лица:**

- Только по заявке от организатора
- Ручная проверка ИНН через ФНС

**Блокировка:**

- Без удаления данных
- Заморозка активных заявок

**Сброс пароля:**

- Через подтверждение ИНН + E-mail
- Временный пароль (действует 24 часа)

**Отчеты по пользователям:**

- Активность (дата последнего входа)
- Количество заявок за период
- Сумма оплат
