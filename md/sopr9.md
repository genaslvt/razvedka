# 9. Модификация отчетов

**Типы отчетов (поддерживаются):**

- Сводка по закупке (лоты, участники, суммы)
- Статусы оплат по юр. лицам
- История заявок за период
- Не достигнутые MOQ

**Сроки разработки:**

| Изменение | Время |
|-----------|-------|
| Новый отчет | 3 рабочих дня |
| Добавить колонку | 1 день |
| Изменить фильтр | 4 часа |

**Форматы выгрузки:** XLSX, CSV, PDF

**Как заказать:** тикет в системе с примером (mockup)
